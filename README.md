# Bank Customer Churn Analysis

## Project Overview
This project explores customer churn behavior in a banking dataset using **Python and data analysis techniques**. The goal of the analysis is to identify patterns that explain why customers leave the bank and highlight key factors that influence churn.

## Project Objectives
- Analyze customer characteristics related to churn.
- Identify patterns between churned and retained customers.
- Provide insights that could help banks improve customer retention.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- The overall churn rate in the dataset is **approximately 12.19%**, meaning about **13 out of every 100 customers leave the bank**.
- **Number of Products (NumOfProducts)** was the strongest churn indicator. Customers with only **one product** showed the highest churn rate, suggesting lower engagement with the bank.
- **Balance** showed moderate influence on churn behavior.
- **Age and Credit Score** did not show strong differences between churned and retained customers.

## Conclusion
The analysis suggests that **customer engagement**, particularly the number of banking products owned, plays a major role in customer retention. Encouraging customers to adopt multiple banking products may help reduce churn.
